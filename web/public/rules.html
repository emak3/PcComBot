コミュニティルール

みんなが快適に過ごすためのルールです。必ずお読みください。

重要事項

上記のルールを破った場合、BANされる前に数回の警告をしますが、重度の違反行為や明らかな荒らし行為は警告なしに即BANとなる場合があります。

DiscordのTOS に違反した場合、即座にBANされます。上記以外の行為であっても、運営は適切と思われる処置を行う権利があります。

スパムは禁止です！

以下の行為は禁止されています：

  • 短縮URLや隠しURL
  • 過剰な絵文字
  • 過剰なリアクション
  • テキストウォール（長すぎる文章）
詳細説明

スパム行為は他のメンバーの体験を損ねる行為です。適度な絵文字の使用や通常の会話は問題ありません。

SFWを心がけましょう！

常識の範囲内であれば、多少の悪口は問題ありません。

もしあなたのユーザー名 / PFPが不快なものであれば、変更を求められる場合があります。

用語説明

SFW: Safe for work（職場でも安全な内容）

PFP: Profile picture（プロフィール画像）

避けるべき内容

過度に攻撃的な言葉、不適切な画像、職場で見るのが困難な内容

適切な内容

建設的な議論、適度な冗談、誰でも見ることができる内容

読みやすい名前にしましょう！

メンションがしにくい特殊文字を含んだユーザー名の場合は変更が必須です。

使用可能な文字

  • 英語
  • 英数字
  • 漢字
  • カタカナ
  • ひらがな
  • 一部記号

制限事項

  • 名前の初めに記号は入れられません
  • 特殊文字を含むことはできません
  • メンション時に困難な文字は避けてください

使用可能な記号

[ ] @ ^ $ ( ) _ { } | ? + \ * .
理由

読みやすい名前により、コミュニケーションが円滑になり、メンションやメッセージのやり取りがスムーズになります。

一般的な会話のみ

宗教、政治、その他ネガティブで攻撃的な話題は禁止です。

論争を始めようとすることは、攻撃的であるとみなされます。

禁止トピック

  • 宗教的な話題
  • 政治的な話題
  • 攻撃的な内容
  • 論争を誘発する内容
  • 差別的な発言
  • 過度にネガティブな内容
理由

PCコミュニティは技術的な学習と情報交換を目的としています。建設的で前向きな環境を維持するため、論争を招く話題は避けています。

広告や自己宣伝は禁止

許可された場合以外や、話の流れにそぐわない自分のコンテンツやチャンネルのリンクは禁止です。

また、Discord等のコミュニティー勧誘は一切認めません。

禁止例

  • 無関係なYouTubeチャンネル宣伝
  • 他のDiscordサーバーへの勧誘
  • 商品やサービスの宣伝
  • アフィリエイトリンク
  • 営利目的の投稿
例外について

技術的な質問への回答や情報共有の一環として、関連するリンクを共有することは問題ありません。ただし、主目的が宣伝でないことが重要です。

質問はガイドを読んでから！

質問フォーラムを使用する前に、必ず質問ガイドラインを読んでください。

明らかに読んでいないと分かる質問に回答する行為も禁止です。

不足部分は回答側が質問する前にスレッドに投稿される埋め込みの選択肢から選択してください。

質問ガイドラインの重要性

適切な質問により、より早く正確な回答を得ることができます。回答者の時間を尊重し、効率的な問題解決につながります。
